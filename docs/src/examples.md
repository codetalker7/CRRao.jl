## Examples: Setting up the code

```@repl examples
using RDatasets, NLSolversBase, CRRao, Logging, StableRNGs;

Logging.disable_logging(Logging.Warn); CRRao.setprogress!(false);

CRRao.set_rng(StableRNG(123))

```

